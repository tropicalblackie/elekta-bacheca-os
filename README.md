# Elekta Bacheca OS v8

CRM immobiliare per la gestione della pipeline deals, discovery di mercato e agenda commerciale.

## Funzionalità principali

- **Bacheca Deals** — Pipeline kanban/tabella con KPI, filtri, export CSV
- **Discovery Mercato** — Analisi di zona con cantieri comparabili e prezzi obiettivo
- **Agenda Commerciale** — Cruscotto operativo con follow-up e proposte in attesa
- **Persistenza localStorage** — I dati restano tra le sessioni
- **Gestione Agente** — Vista commerciale con log note e tag

## Avvio locale

```bash
npm install
npm run dev
```

## Build produzione

```bash
npm run build
npm run preview
```

## Stack

- React 18 + Vite
- Tailwind CSS
- lucide-react
